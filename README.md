**XChangeEase** is a simple and responsive currency converter web app built using **HTML, CSS, and JavaScript**.  
It fetches real-time exchange rates from the [ExchangeRate API](https://www.exchangerate-api.com/) and lets users easily convert between global currencies with country flags and instant results.

---

## 1. Features

1. Real-time currency exchange rates  
2. Converts instantly between 150+ currencies  
3. Auto-updating country flags  
4. Clean, responsive, and beginner-friendly UI  
5. Built with Fetch API and async/await

---

## 2. Tech Stack

1. **Frontend:** HTML, CSS, JavaScript  
2. **API:** [ExchangeRate API](https://www.exchangerate-api.com/)  
3. **Hosting:** [Vercel](https://vercel.com)
